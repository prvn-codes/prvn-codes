### Open source

**[esctl](https://github.com/prvn-codes/esctl)** — CLI to manage multiple local Elasticsearch clusters. Start, stop, list, import and inspect — without touching config files manually.

```sh
esctl start myapp
esctl list
esctl logs -f myapp
```
